Uppercase IBM Plex Mono kicker used above headings and as the small label on cards, metrics and metadata. Follows the accent color (clinical teal-blue, theme-aware).

```jsx
<Eyebrow rule>Hexaros — Tecnologie per la farmacia</Eyebrow>
<Eyebrow>I nostri prodotti</Eyebrow>
```

Set `rule` for the short leading accent line seen in hero labels. Use `as` to render a `div`/`p` when needed.
